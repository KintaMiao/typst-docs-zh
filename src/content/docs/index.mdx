# 概述
欢迎来到 Typst 的文档！Typst 是一个基于标记语言的全新排版系统，
专为科学领域设计。它旨在成为高级工具（如 LaTeX）和简单工具（如 Word 和 Google Docs）的替代选择。我们构建 Typst 的目标是
打造一个功能强大且使用愉快的排版工具。

本文档分为两个部分：一个是对初学者友好的教程，
通过实际用例介绍 Typst；另一个是全面的参考手册，
解释 Typst 的所有概念和功能。

我们还邀请您加入我们围绕 Typst 构建的社区。Typst 仍然是一个非常年轻的项目，
因此您的反馈意见非常宝贵。
